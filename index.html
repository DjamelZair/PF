Woningbouw Plannen Amsterdam Dashboard

This interactive dashboard shows where and when new housing projects are planned in Amsterdam. Built with Dash, it integrates a public API, geospatial transformation (RD ‚Üí WGS84), and interactive/dynamic visuals like heatmaps and charts.

Users can explore housing density, project timelines, and neighborhood distributions using filters for housing type and year.

Keywords: Dash, Plotly, REST API integration, geolocation (RD/WGS84), data wrangling, interactive visualizations, sliders, checklists, map overlays, data pipelines

Bachelor Thesis: From Coordinates to Real Estate Insights

A RAG-Based Pipeline for Structured Real Estate Data

This thesis (grade: 8/10) introduces a Retrieval-Augmented Generation (RAG) pipeline to enrich housing project polygons with metadata: address info, investment parameters, nearby amenities (banks, schools, supermarkets), and multilingual support.

I scraped public datasets (via Google + static sites), queried OSM/Nominatim, and performed hybrid reranking to compare OpenAI GPT models and Mistral 7B. My system returned context-aware summaries for planning stakeholders.

Skills: Python, REST APIs, web scraping, multilingual NLP, geospatial enrichment, RD‚ÜîWGS84, hybrid reranking, OpenAI API, Mistral 7B, JSON/GeoJSON, vector search, amenity classification
